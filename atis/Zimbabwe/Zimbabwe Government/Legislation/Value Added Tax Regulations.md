# Value Added Tax Regulations

**Status**: Active
**Enforced By**: 
  - "[[Zimbabwe Revenue Authority (ZIMRA)]]"


## Description
Controls corporate standard VAT filings, import VAT exclusions, and zero-rated categories, fundamentally affecting short-term liquidity management on imported capital mining equipment.
